================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Failed to start Eliza server"
Solution: 
  - Run: bun i -g @elizaos/cli
  - Check that trading-brain directory is accessible
  - Verify port 3000 is not in use

Problem: "Cannot find trading-brain project"
Solution:
  - Set environment variable: ELIZA_PROJECT_PATH
  - In PowerShell: $env:ELIZA_PROJECT_PATH="C:\path\to\trading-brain"
  - In CMD: set ELIZA_PROJECT_PATH=C:\path\to\trading-brain

Problem: Server takes too long to start
Solution:
  - Check that Ollama is running (if using local models)
  - Verify .env file is configured
  - Check console for error messages
